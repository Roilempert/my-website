עקבות — HOW TO OPEN THE SITE ON THE iMac
=========================================

PRESENTATION MACHINE
--------------------
21.5-inch iMac — the site is built and tuned for this screen at the exhibition.


>>> EASIEST WAY — NO SERVER, NO TERMINAL, NO PYTHON <<<
========================================================
The site now runs straight from a file. Nothing to install.

STEP 1 — OPEN THE PROJECT FOLDER
  Double-click the folder:  my-website

STEP 2 — DOUBLE-CLICK THE START FILE
  Double-click:  opening.html

  It opens in the default browser (the ceremonial opening screen).
  Click  כניסה  to enter the experience.

That's it. The note data is baked into the files (data/embedded-data.js),
so the site works fully offline with no local server.

TIP — open in Chrome, full screen
  If opening.html opens in Safari and you prefer Chrome:
  RIGHT-CLICK opening.html → Open With → Google Chrome.
  Then press  Cmd+Ctrl+F  for full screen.

NOTE — this build is NOT live
  Data is a frozen snapshot, not the live Google Sheet. To refresh the
  snapshot after editing the sheet/CSVs, on a Mac with Node run:
      ./build-embedded-data.sh
  (Only needed by whoever updates the content — not at the exhibition.)


---------------------------------------------------------
OPTIONAL — the old local-server launcher (only if needed)
---------------------------------------------------------
The double-click method above replaces this. Use the server only if you
specifically want the live Google Sheets data.

Double-click (or right-click → Open the first time):

  Start Local Server.command      (or  Start Site.app)

A Terminal window opens by itself; wait for:

  Site is ready: http://127.0.0.1:8080/opening.html

The launcher tries Python, then Ruby, then Perl. If it says all three are
missing, that Mac needs Xcode Command Line Tools — but you do NOT need any
of this: just use the double-click method above instead.


IF macOS BLOCKS A FILE (first time only)
----------------------------------------
Right-click the file → Open → click Open in the dialog.


STEP 4 — IF THE BROWSER IS EMPTY
--------------------------------
Wait 10 seconds. Use the browser menu: View → Reload Page.


TO STOP
-------
Close the Terminal window that opened.


ATTRACT MODE (exhibition)
-------------------------
After about 90 seconds with no touch, the site runs a short automatic
demonstration. Touch anywhere to take over.


DEV NOTE (not for exhibition iMac)
----------------------------------
Exhibition opens: http://127.0.0.1:8080/  (redirects to opening.html)

Skip opening during dev (one visit):
  http://127.0.0.1:8080/opening.html?skipOpening=1

Test show reel (skip idle, start demo on load):
  http://127.0.0.1:8080/experience.html?showReel=autostart

Disable show reel:
  ?showReel=0

Main site directly:
  http://127.0.0.1:8080/experience.html


NEED TO COPY AGAIN?
-------------------
If you copied the folder before these files existed, copy the whole
my-website folder again from the USB / other Mac. You need:

  opening.html
  experience.html
  index.html               (redirect → opening)
  data/embedded-data.js    (baked data — required for no-server mode)
  js/offline-data-shim.js  (serves baked data under file://)
  data/main.csv
  data/tags.csv
  data/opening-palette.json
  js/  vendor/  styles.css  assets/
  Start Site.app                 (optional — live-data server)
  Start Local Server.command     (optional — live-data server)
