עקבות — HOW TO OPEN THE SITE ON THE iMac
=========================================

You will NOT find a start button in the folder right-click menu.
Follow these steps in Finder:


STEP 1 — OPEN THE PROJECT FOLDER
--------------------------------
Double-click the folder:  my-website

(You must go INSIDE the folder first.)


STEP 2 — IF macOS SAYS "UNIDENTIFIED DEVELOPER"
-----------------------------------------------
Read:  READ ME FIRST — macOS security.txt

Quick fix (mouse only):
  RIGHT-CLICK  Start Local Server.command  →  Open  →  Open


STEP 3 — START THE SITE
-----------------------
Double-click (or right-click Open, first time only):

  >>>  Start Local Server.command  <<<

(A Terminal window opens by itself. You do NOT need to type anything.)

Wait until you see:

  Site is ready: http://127.0.0.1:8080/

Then Chrome opens automatically. Leave Terminal open.


Alternative: double-click  Start Site.app  (does the same thing)


IF YOU SEE "SERVER FAILED"
--------------------------
The new launcher works without the scripts/ folder.

1. Copy the whole my-website folder again from USB
2. Close ALL Terminal windows (red X)
3. RIGHT-CLICK  Start Local Server.command  →  Open  →  Open
4. Read the Terminal window — it now shows the real error

If it says Python/Ruby missing:
  This Mac needs "Command Line Tools" installed once (ask someone with a keyboard).


STEP 3 — IF macOS BLOCKS THE FILE (first time only)
---------------------------------------------------
Right-click the file → Open → click Open in the dialog.

(Do NOT look for it in the folder right-click menu.)


STEP 4 — IF THE BROWSER IS EMPTY
--------------------------------
Wait 10 seconds. Use the browser menu: View → Reload Page.


TO STOP
-------
Close the Terminal window that opened.


NEED TO COPY AGAIN?
-------------------
If you copied the folder before these files existed, copy the whole
my-website folder again from the USB / other Mac. You need:

  Start Site.app
  Start Local Server.command
  data/main.csv
  data/tags.csv
  index.html
  js/  vendor/  styles.css  assets/
